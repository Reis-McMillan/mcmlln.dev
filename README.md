# mcmlln.dev

Personal portfolio website for Reis McMillan.

## Tech Stack

- [Nuxt 4](https://nuxt.com) + [Vue](https://vuejs.org)
- [Tailwind CSS v4](https://tailwindcss.com)
- [GSAP](https://gsap.com) + [Lenis](https://lenis.darkroom.engineering/) for animations
- [Reka UI](https://reka-ui.com) for accessible components

## Development

```bash
npm install
npm run dev
```

Requires Node.js v22+.
